emfont 是一款免費、簡單易用的 webfont 服務， 讓你只需在網站內加上兩行 JavaScript，並在元素上套用 class，就能立即套用豐富的字型。 不論是作品集、部落格還是專業平台，都能大幅提升視覺質感。

目前 emfont 提供數十款開源中文字型，共上百種不同字重選擇，輕巧、穩定且免授權煩惱。 不必處理繁瑣的設定，也不需擔心版權問題，讓設計和開發專注於創意本身。

本專案特別感謝 Zeabur 以及 國雲網路 提供台灣高速穩定的運算資源。 讓 emfont 能持續為大家提供優質服務。而最重要的，是每一位使用者的支持——我們的使命，就是讓設計師不再受限於中文字型，盡情綻放腦中的創意。

如果你喜歡這個開源專案，歡迎在 GitHub 留下一顆星星。 而如果你有任何問題或有興趣協助開發，不要吝嗇在我們的 Discord 伺服器 中提出。

第一次使用 emfont？

emfont 是一個免費的網頁字體 (Webfont) 服務，會自動壓縮與最佳化字型檔案。你可以 閱讀文檔來快速入門。

文字是網站的靈魂，是思想的表達，更是美感的載體。然而中文字體因為檔案龐大、需要耗費大量流量，很少有免費中文字體的 webfont 服務。emfont 的誕生正是為了填補這一空白，整理了許多開源免費的字體。為開發者和設計師們提供一個優秀的中文字體解決方案。
